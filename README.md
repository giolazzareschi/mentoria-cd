# Mentoria CD

## Comandos basicos do git:
- git init: inicia um repositorio local
- git add README.md: adiciona arquivos para serem versionados
- git commit -m "first commit": um texto para descrever o que foi feito na alteracao
- git branch -M main: selecionar ou criar branches de trabalho
- git remote add origin git@github.com:giolazzareschi/mentoria-cd.git : apontar para onde vai ser conectado (quem é a origin - nuvem)
- git push -u origin main: de fato enviar
- git status: ver quais arquivos foram modificados
